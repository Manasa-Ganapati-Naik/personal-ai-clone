
### In VS Code:
1. Right-click the repo root → “New File” → `CONTRIBUTING.md`
2. Paste the following:

```markdown
# Contributing to Personal AI Clone

Thank you for considering contributing! Please follow these guidelines.

## Code Style
- Python: [PEP 8](https://www.python.org/dev/peps/pep-0008/)
- JS/React: [Airbnb style guide](https://github.com/airbnb/javascript)
- Docstrings: Use Google style for Python functions
- Commit messages: Use conventional commits:
  - `feat:` for new features
  - `fix:` for bug fixes
  - `docs:` for documentation
  - `chore:` for project maintenance
  - `refactor:` for code refactoring

## Branching
- Use feature branches: `feature/<short-description>`
- PRs should be reviewed before merging to `main`

## Reporting Issues
- Use GitHub Issues to report bugs or request features
- Include clear steps to reproduce and relevant screenshots/logs

## Pull Requests
- PR titles should be concise, e.g., `feat: add text generation route`
- Reference related issues in the PR description
- Ensure tests pass before merging

## Code Review
- Ensure code is readable, modular, and well-commented
- Follow existing project structure
